Player Zone - Plataforma de Jogos Digitais

Sobre o Projeto

O Player Zone é uma plataforma web desenvolvida com o objetivo de simular uma loja virtual de jogos digitais, proporcionando uma experiência de navegação moderna, organizada e responsiva.
O projeto foi construído utilizando HTML5, JavaScript e Bootstrap 5, com foco em responsividade e usabilidade em diferentes dispositivos.

Objetivo

O principal objetivo do sistema é permitir que o usuário:

- Navegue por um catálogo de jogos organizado por categorias
- Visualize informações detalhadas de cada jogo
- Simule um sistema de compras através de um carrinho de compras
- Realize login e cadastro de forma simples (simulação front-end)

Funcionalidades do Sistema

O site conta com as seguintes funcionalidades:

- Página inicial com catálogo de jogos
- Sistema de login e cadastro (interface)
- Separação por categorias:
- Jogos de Ação
- Jogos de Corrida
- Jogos de Esportes
- Carrinho de compras com armazenamento local (localStorage)
- Sistema de busca de jogos
- Modais com informações detalhadas dos jogos
- Layout totalmente responsivo

Tecnologias Utilizadas

- HTML5
- JavaScript
- Bootstrap 5

Interface e Experiência

O design do projeto foi pensado para simular uma plataforma real de games, utilizando:

- Tema escuro para melhor imersão
- Cards para organização dos jogos
- Imagens em destaque para cada produto
- Navegação intuitiva e rápida
- Estrutura responsiva para mobile, tablet e desktop


Estrutura de Desenvolvimento

O projeto foi dividido em múltiplas páginas para melhor organização:

- index.html : Página principal (catálogo)
- login.html : Login do usuário
- cadastro.html : Cadastro de usuário
- páginas de categorias : ação, corrida e esportes.

Durante o desenvolvimento do projeto, foi utilizada uma ferramenta de Inteligência Artificial como apoio para:

- Sugestão de melhorias no layout
- Organização da documentação
- Apoio na estruturação de código JavaScript
- Revisão de boas práticas de desenvolvimento

Mesmo com esse suporte, todo o projeto foi implementado e ajustado manualmente, garantindo o entendimento dos conceitos trabalhados em aula.
